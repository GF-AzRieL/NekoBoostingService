This project is a **satirical landing page** for a fictional "boosting" service dedicated to the game **Strinova**. Built for the community and for laughs, it features a high-end Cyberpunk/Anime aesthetic inspired by the game's "Calabiyau" UI.

---

## 🚀 Key Features

* **Responsive & Immersive Design**: A neon color palette (Calabiyau Cyan and Alert Orange) featuring glassmorphism and complex geometric shapes using CSS `clip-path`.
* **Dynamic Booster Profiles**: An interactive section allowing users to toggle between various legendary "boosters" (Neko, Elabearie, Junlean, etc.) using a custom JavaScript tab system.
* **Animated Client Reviews**: A dual-lane marquee system with reverse-scrolling testimonials to simulate high traffic and (ironic) credibility.
* **Custom Order Modal**: A sleek pop-up form for "hiring" services, including Discord ID validation and region selection.
* **Satirical Service Packages**: Offers everything from "Hard Carry" Singularity runs to the "Safest Derank" for those who want to gatekeep the lower tiers.

## 🛠️ Technical Stack

* **HTML5**: Semantic structure and layout.
* **CSS3**: Custom keyframe animations (`scrollLeft`, `bounce`), root variables for easy theming, and modern Grid/Flexbox layouts.
* **JavaScript (Vanilla)** : Handles tab logic, modal state management, and the simulated order processing.
* **External Assets**: Integrated FontAwesome 6 icons and Google Fonts (Rajdhani & Inter).
